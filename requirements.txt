## Requirements

- Python 3.8 or higher

## Installation

### 1. Clone the repository
```bash
git clone https://github.com/yourusername/brawler-game.git
cd brawler-game
```

### 2. Create virtual environment (recommended)
```bash
python -m venv venv

# On Windows
venv\Scripts\activate

# On macOS/Linux
source venv/bin/activate
```

### 3. Install dependencies
```bash
pip install -r requirements.txt
```

## Running the Game

```bash
python main.py
```

Or use the runner script:
```bash
python run.py
```

## Controls

### Player 1
- **A/D**: Move left/right
- **W**: Jump
- **R**: Attack 1
- **T**: Attack 2

### Player 2 (Multiplayer Mode)
- **Arrow Keys**: Move left/right
- **Up Arrow**: Jump
- **Numpad 1**: Attack 1
- **Numpad 2**: Attack 2

## Adding New Characters

To add a new character, update the `CHARACTERS` dictionary in `main.py`:

```python
CHARACTERS = {
    "your_character": {
        "name": "Character Name",
        "path": "assets/images/characters/your_character/sprite.png",
        "data": [size, scale, [offset_x, offset_y]],
        "steps": [idle, run, jump, attack1, attack2, hit, death]
    }
}
```

## Game Assets

Place your game assets in the following structure:
- **Character sprites**: `assets/images/characters/{character_name}/Sprites/`
- **Backgrounds**: `assets/images/background/` (supports .gif)
- **Music**: `assets/audio/music.mp3`
- **Sound effects**: `assets/audio/*.wav`
- **Fonts**: `assets/fonts/*.ttf`
